# Repository instructions

## Local Japanese writing skill

When the user asks for natural Japanese, human-written feel, anti-AI cleanup, rewriting, polishing, style adjustment, or says 「このskills使って」, use the local skill `ja-natural-humanizer` from `.agents/skills/ja-natural-humanizer/`.

Prefer this skill for:
- Japanese emails
- internal docs
- explanatory prose
- blog posts
- summaries
- release notes
- rewrites of existing Japanese drafts

Do not use it as the primary workflow for:
- literal translation
- contracts, terms, policies, 約款
- patent claims
- intentionally promotional or formulaic copy

When revising Japanese text:
- preserve facts, numbers, names, scope conditions, and caveats
- simplify wording only when meaning stays intact
- return the finished Japanese text first unless the user explicitly asks for analysis
