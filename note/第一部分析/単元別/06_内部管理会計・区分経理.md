# 内部管理会計・区分経理

## この資料の狙い

- 第7章を `法定会計の限界 -> 経営用の物差し -> 商品別/区分別管理` の一本の流れで読めるようにする
- `EV` `MCEV` `区分経理` を別々の暗記事項ではなく、`経営者に何を見せたいか` をめぐる同じ章として整理する
- 第I部で混ざりやすい `法定会計と内部管理会計` `伝統的EVとMCEV` `商品区分と全社区分` の違いを先にほどく
- 区分経理を、`配当の公平性` だけでなく `収益管理` `ALM` `商品設計` `リスク管理` にどう効くかまで押さえる

## 教科書との対応

教科書第7章は、大きく見ると次の順で進む。

1. 内部管理会計の意義と必要性
2. EV とアプレイザル・バリュー
3. ハードルレート、ROI、資本コスト
4. 伝統的EV と MCEV の違い
5. 区分経理の意義
6. 商品区分・全社区分・資産区分の考え方
7. 区分経理の活用方法と留意点

一見すると、前半は価値評価、後半は監督指針という別々の話に見えやすい。  
でも本当は全部、`法定会計だけでは経営に必要な情報が足りない` という問題から出ている。

生命保険会社では、

- 新契約を伸ばしても単年度利益は悪く見えうる
- 商品ごとの収益性が会社全体の数字に埋もれやすい
- 金融市場や契約者行動が変わると、法定会計の見え方と経済実態がずれる
- 配当の公平性、セルフサポート、資産運用方針を商品群ごとに見ないと判断を誤る

という事情がある。  
第7章は、このずれを埋めるために `管理用の会計` と `区分して見る仕組み` を置く章だと読むと、かなり整理しやすい。

## まず、この章は何を解決したいのか

生命保険会社は、契約獲得から給付支払いまでの時間が長い。  
そのため、法定会計の単年度利益だけを見ても、経営が本当に良くなっているのか悪くなっているのかが見えにくい。

たとえば、

- 新契約獲得時には募集手数料や新契約費が先に出るので、単年度利益は悪く見えやすい
- 解約時には解約控除の影響で利益がよく見えることがある
- 保険種類やチャネルごとの収支構造の違いが、会社全体の数字に埋もれやすい
- 商品特性が違うのに一括管理していると、どこが稼いでどこが資本を食っているのか見えにくい

といったことが起こる。

ここで必要なのが、`監督用の会計` と `経営用の会計` を分けて考える視点である。  
監督や契約者保護のために保守的な会計は必要だが、それだけでは経営判断には足りない。  
そこで、商品別、チャネル別、資産区分別に情報を切り出し、必要なら将来キャッシュフローや経済価値まで見にいく。

この章を一本で言うと、

1. 法定会計では見えにくい経営実態を補う
2. 将来利益や経済価値を読む物差しを入れる
3. 区分経理で公平性と収益管理を両立させる

という章である。

## 教科書本文の節ごとの読み方と第I部想定問題

ここでは、第7章を `この節は何を見えるようにしたいのか` から読む。

### 7.1 内部管理会計の意義と必要性

この節の出発点は、法定会計が悪いという話ではない。  
法定会計は、契約者保護や監督、一般投資家への情報提供に向いた会計である。  
ただ、そのままでは経営者が知りたい `現在の収益力` `商品別の収支構造` `投入資本に対するリターン` を十分には示してくれない。

だから内部管理会計では、

- 会社の経済的基盤を反映すること
- プロフィットセンターごとに結果が見えること
- 上層部が理解しやすいこと

が求められる。

- 第I部想定問題: 内部管理会計の意義および必要性を説明せよ。
- 第I部想定問題: 現行法定会計の特徴と限界に触れつつ、内部管理会計が必要な理由を述べよ。

### 7.2 EV とアプレイザル・バリュー

ここは、`今ある純資産だけでは保険会社の価値を読み切れない` ことを補う節である。  
保険会社の価値は、保有契約から将来生まれる利益まで含めて見ないと実態に近づかない。  
そこで EV が出てくる。さらに、将来の新契約獲得余地まで含めようとするとアプレイザル・バリューになる。

この節では、`保有契約由来の価値` と `将来ののれん的価値` を切り分けることが大事である。

- 第I部想定問題: EV とアプレイザル・バリューを簡潔に説明せよ。
- 第I部想定問題: 潜在価値会計の意義、考え方、特徴を述べよ。

### 7.3 ハードルレート、ROI、資本コスト

ここは、将来利益を現在価値に引き直すときの物差しの節である。  
単に利益が出るかではなく、`投入した資本に見合う収益率か` を見ないと経営判断には使いにくい。  
その比較軸が ROI であり、要求される収益率がハードルレートであり、拘束資本が稼げなかった分が資本コストになる。

この節は、価値評価が `利益の大きさ` だけではなく `資本の機会費用` を含むことを押さえる節と読むと分かりやすい。

- 第I部想定問題: ハードルレートについて説明せよ。
- 第I部想定問題: トップダウンアプローチとボトムアップアプローチの違いを述べよ。
- 第I部想定問題: 潜在価値会計における資本コストを説明せよ。

### 7.4 伝統的EV と MCEV

この節は、`リスクをどこまで明示的に反映するか` が中心である。  
伝統的EVはハードルレートにかなり依存してリスクを織り込む。  
MCEV は市場整合的評価を使い、TVOG、フリクショナル・コスト、残余ヘッジ不能リスクを切り出して見る。

大事なのは、MCEV が `きれいな高級版EV` なのではなく、`リスクの置き場所をはっきりさせた評価枠組み` だということだ。

- 第I部想定問題: 伝統的EV と MCEV の主な相違点を述べよ。
- 第I部想定問題: TVOG、フリクショナル・コスト、残余ヘッジ不能リスクを説明せよ。

### 7.5 区分経理の意義

この節は、`損益をどの単位で見るか` の節である。  
配当の公平性、内部補助の遮断、事業運営の効率化、商品設計や価格設定面での創意工夫を図るには、一般勘定を一括で見ているだけでは足りない。

区分経理は単なる事務上の区分ではない。  
商品特性、資産運用特性、チャネル特性の違いを踏まえて、どの群団がどのように収益を生み、どのようにリスクを抱えているかを見るための管理インフラである。

- 第I部想定問題: 区分経理の意義を説明せよ。
- 第I部想定問題: 区分経理が保険計理人の確認業務とどう関係するか述べよ。

### 7.6 商品区分・全社区分・資産区分

ここは、区分経理を実務に落とした節である。  
商品区分は、損益と負債を把握する単位として合理的に設定される必要がある。  
全社区分は、リスクバッファー、新商品開発資金、共通資産管理、現預金管理などを担う。  
資産区分は、商品特性や ALM の必要に応じて資産運用を対応づける。

ここで重要なのは、`細かく分ければよい` ではないことだ。  
小さすぎる区分は分散効果を失い、保険収支や運用効率を不安定にする。  
区分は、経営判断に役立つ粒度で切る必要がある。

- 第I部想定問題: 商品区分の設定における留意点を説明せよ。
- 第I部想定問題: 全社区分の機能を挙げよ。
- 第I部想定問題: 商品区分と全社区分との取引を分類して説明せよ。

### 7.7 区分経理の活用方法と留意点

最後の節は、区分経理を何に使うかの節である。  
配当の公平性確認、セルフサポート確認、不採算区分の見直し、販売政策、ALM、商品設計、利源分析、新商品開発と、使い道は多い。

ただし、使い道が多いほど、事業費配賦の精度、システム負荷、細分化しすぎの問題、経営陣の理解不足といった副作用にも注意が必要になる。

- 第I部想定問題: 区分経理の活用方法と留意点を説明せよ。
- 第I部想定問題: 区分経理を活用するうえで、事業費配賦や費用対効果にどのような留意が必要か述べよ。

## 法定会計ではなぜ経営判断に足りないのか

ここは第7章の土台なので、答案でも最初に一言で言えると強い。

法定会計は、

- 支払能力の確保
- 契約者保護
- 監督当局への報告
- 一般投資家への情報提供

には向いている。  
しかし生命保険会社の経営者が知りたいのは、`いま何が本当に儲かっているのか` `どこで資本を使いすぎているのか` `どの商品が将来価値を作っているのか` である。

そのため、法定会計だけでは、

- 新契約獲得時の見え方
- 解約時の見え方
- 商品別・チャネル別の収支構造
- 将来利益やリスクの織込み方

が経営判断向けには粗い。

このため、内部管理会計は `法定会計を否定する会計` ではなく、`法定会計を補う経営用の会計` と読むのが正しい。

## EV と MCEV は何を補っているのか

EV 系の論点は、`区分経理とは別の話` に見えやすい。  
でも実際には、どちらも `単年度損益だけでは見えない価値を見にいく` という点でつながっている。

EV は、保有契約から将来出る期待利益と純資産を組み合わせて、生命保険会社の経済的価値を見ようとする。  
MCEV はそこからさらに、最低保証や契約者オプション、市場リスク、ヘッジ不能リスク、資本維持コストまで、より明示的に扱おうとする。

第I部で差がつきやすいのは、次の切り分けである。

- 伝統的EV: ハードルレート中心でリスクを織り込む
- MCEV: 市場整合的評価を土台にしてリスクの項目を切り出す
- アプレイザル・バリュー: 保有契約価値に将来新契約のポテンシャルを足す

この3つを混ぜないようにしたい。

## 区分経理は公平性の制度であり、経営管理のインフラでもある

区分経理というと、まず配当の公平性が出てくる。  
もちろんそれは正しい。  
有配当契約と無配当契約、保障性商品と貯蓄性商品、固定金利商品と変動型商品などを同じ箱で処理すると、どの契約がどの剰余を生んだのかが見えにくくなり、公平な利益還元が難しくなる。

ただ、現代の試験で差がつくのはそこから先である。  
区分経理は、

- セルフサポートの確認
- 不採算区分の発見
- 商品設計や予定率設定への反映
- ALM の単位設定
- 商品別、チャネル別の収益管理
- 新商品開発時の事業運営資金管理

までつながる。

つまり区分経理は、`契約者間の公平性を守る制度` であると同時に、`経営判断に使うデータの入れ物` でもある。

## この章で混ざりやすい論点

### 法定会計と内部管理会計

- 法定会計は監督・契約者保護・外部報告向け
- 内部管理会計は経営判断向け
- 両者は対立概念ではなく、目的が違う

### 伝統的EV と MCEV

- 伝統的EVはハードルレートによりかなり一括してリスクを織り込む
- MCEV は市場整合的評価、TVOG、フリクショナル・コスト、残余ヘッジ不能リスクを明示する
- MCEV でも必要資本は消えないので、`資本コストが全く不要` と短絡しない

### 商品区分と全社区分

- 商品区分は損益・負債を把握する単位
- 全社区分はバッファー、共通資産、現預金、事業運営資金の受け皿
- どちらも必要で、片方だけでは区分経理は機能しない

### 細分化と有効性

- 細かく分けると実態に近づく面はある
- ただし分散効果の低下、事務負荷、配賦の不安定化がある
- だから `理論的・合理的で、かつ経営判断に役立つ粒度` が必要になる

## 第I部でこの章をどう持っていくか

この章は、定義だけ覚えても崩れやすい。  
覚え方としては、次の順が安定しやすい。

1. 法定会計ではなぜ足りないのか
2. だから何を見えるようにしたいのか
3. そのための道具が EV/MCEV と区分経理である
4. 道具には目的ごとの長所短所がある

この順で腹に入れておくと、長めの説明問題でも軸がぶれにくい。
