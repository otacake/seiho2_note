# 生保2 過去問丸暗記

このディレクトリは、WB と H23-2025 の過去問を単元別に並べ、原文ベースで白紙再現するための正本です。

## 使い方

1. `deep_input/` で単元の芯と制度の意味を読む。
2. `units/` で問題の意味と背景を確認する。
3. Anki の `whole` カードで、原問題だけを見て暗記答案を白紙再現する。
4. 長い答案は `chunk` カードで、観点・段落ごとに原文を固める。
5. `cloze` は数値、条文語句、列挙名、制度名だけに使う。

## 今回の再設計ルール

- 長すぎる答案、複数小問の羅列、答えが問題面に残りやすいカードは、`whole` を答案骨子カードにし、原文再現は `chunk` へ逃がす。
- `chunk` は見出し・列挙・小問単位で切り、1枚に複数観点を詰め込まない。
- `cloze` は数値、条文、制度名、列挙名、定義語句に限定し、文章全体が読めなくなる抜き方はしない。
- 小問カードで覚えるべき語句・処理・理由が黒字に残る場合は、問題文として必要な文脈だけを残し、答え側を穴埋めにする。
- 既存デッキへ上書きインポートすると、消えた旧カードがAnki側に残ることがある。再取り込み時は `生保2::過去問丸暗記` デッキを削除してから、このAPKGを入れる。

## 要約禁止ルール

- `暗記答案` は、公式解答例・WB・既存単元別Markdownの文言を原則保持する。
- 許可する加工は、OCRノイズ除去、ページヘッダ削除、改行・箇条書き整形、明らかな文字化け補正だけ。
- 意味説明、背景、言い換えは `units/` の `理解メモ` に置き、Anki の back へ混ぜない。
- `whole` が答案骨子カードになっているものは、暗記原文そのものではなく、分割カードへ進むための索引として扱う。原文は `chunk` と `units/` に保持する。

## 生成方法

```powershell
python scripts/build_past_exam_memory.py
python scripts/export_past_exam_memory_apkg.py
```

## 出力

- `past_exam_index.csv`: WB + H23-2025 の小中問台帳
- `deep_input/*.md`: 暗記答案の前提・制度の意味・具体的な読み方を説明する入力教材
- `units/*.md`: 単元別の原問題、暗記答案、理解メモ
- `cards/*.md`: Anki exporter 用カード
- `exports/生保2_過去問丸暗記.apkg`: 章別サブデッキ付き APKG
