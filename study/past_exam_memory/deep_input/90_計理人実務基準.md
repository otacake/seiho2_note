# 90_計理人実務基準

## この単元の芯

保険計理人実務基準は、条文名や確認項目を暗記する単元に見える。しかし芯は、会社の判断をアクチュアリーがどう検証するかである。責任準備金、契約者配当、将来収支分析は、会社の利益や商品戦略と強く関係する。会社には、利益を早く出したい、配当を安定させたい、商品を売りたいという事情がある。一方で、契約者保護と将来の支払能力は崩せない。

保険計理人は、この間に立つ。計算をするだけではない。責任準備金が十分か、配当が公正・衡平か、将来収支で健全性が保たれるかを、実務基準に沿って確認する。実務基準が試験で問われるのは、保険計理人の確認が、会計、配当、ソルベンシー、商品管理を横断するからである。

日本アクチュアリー会の2026年3月19日版の実務基準は、保険計理人の確認・意見書・附属報告書などの実務上の枠組みを定めている。古い過去問の文言を覚えるときも、現行版とのずれがありうるため、条番号や制度変更は確認しながら扱う。

## 暗記事項は何を言っているか

「責任準備金の確認」は、積立額が計算どおりかだけではない。契約者に対する将来債務を履行するために、保険料積立金、未経過保険料、支払備金などが適切に積まれているかを確認する。ここは生命保険会計と直結する。

「契約者配当の確認」は、配当所要額が出せるかだけではない。代表契約、ネット・アセット・シェア、将来の健全性を通じて、返しすぎていないか、契約者間で不公平になっていないかを見る。ここは契約者配当の単元と直結する。

「将来収支分析」は、現在の財務状態だけでなく、将来の保険収支、資産運用、配当、内部留保を見に行く。静的なソルベンシー指標だけでは見えない弱さを確認するための道具として読む。

## 具体的に見る

会社が有配当契約について配当率を据え置きたいとする。営業上は、配当を下げると契約者の納得感が悪くなる。しかし、将来の利差益が弱くなっているなら、同じ配当を続けると内部留保が減る。保険計理人は、会社の希望だけでなく、代表契約のアセット・シェアや将来収支を見て、配当が妥当かを確認する。

また、責任準備金についても、会社が利益を出したいなら負債を小さく見せる誘因がある。保険計理人の確認は、この誘因に対して、制度上の歯止めとして働く。だから実務基準は、単なる事務手続きではなく、契約者保護の実務上の型である。

## 読み方のコツ

この単元は、次の対応関係で覚える。

1. 責任準備金の確認は、将来債務の履行可能性を見る。
2. 配当確認は、公正・衡平と残存契約群団の健全性を見る。
3. 将来収支分析は、現在の数字が将来も耐えるかを見る。
4. 意見書・附属報告書は、その確認を会社と監督へ説明できる形にする。

保険計理人実務基準だけを孤立して覚えるより、生命保険会計、契約者配当、ソルベンシーの横断ルールとして読む方が残りやすい。
