# 05_事業費

## この単元の芯

事業費は、単なる経費削減の単元ではない。生命保険では、契約を獲得するための費用が先に大きく出る。一方、その費用の回収は、保険料を通じて長い期間に分かれる。だから、事業費を見るときは「いくら使ったか」だけでなく、「どの商品が、どの時点で、どの契約群団から回収するのか」を見る必要がある。

予定事業費、事業費枠、チルメル式、費差損益、商品別収益管理は、全部この時間差の話につながる。新契約費を初年度に大きく使うと、会計上は初年度の損益が悪く見えやすい。しかし、保険料の中には将来にわたって費用を回収する部分が含まれている。問題は、その回収見込みが本当にあるのか、商品別に見てどこまで妥当かである。

金融庁の監督指針でも、保険会社の経営管理は、収益性、リスク、商品特性、内部管理を分けずに見る。事業費の答案が商品別原価管理やモニタリングへ伸びるのは、費用が会社全体で足りているかだけでは、商品ごとの採算や契約者間の負担関係が分からないからである。

## 暗記事項は何を言っているか

「予定事業費」は、保険料に織り込まれた事業費の回収予定である。実際にかかった事業費と予定事業費との差が費差損益になる。費差益が出るなら、予定より少ない費用で運営できたということだが、単純に良いとは限らない。必要な管理やサービスを削った結果なら、将来の契約維持や苦情対応に跳ね返る。

「蔵銀枠」や「チルメル式」は、新契約費をどう見込むかの道具として読む。契約初年度に発生する費用を、保険期間全体で回収すると考えると、初年度の事業費枠をどう置くかが問題になる。ここで出てくる枠の話は、計算式を覚えるだけでなく、初年度費用と将来回収の対応関係として見る。

「商品別原価管理」は、会社全体の事業費だけでは見えない問題を分けるためにある。営業職員チャネル、代理店チャネル、ネットチャネルでは、獲得費用、維持費用、システム費、解約率、契約単価が違う。同じ会社の中でも、商品やチャネルによって費用構造が違うなら、全部を平均で見ると判断を誤る。

## 具体的に見る

新しくネットチャネルで積立保険を売り出す会社を考える。発売初年度は広告宣伝費が大きく出る。商品を知ってもらうために必要な費用だが、保険料に含めた予定事業費がそれをすぐ回収できるとは限らない。初年度だけ見ると費差損が出る可能性がある。

このとき「初年度に赤字だから悪い商品」と決めるのは早い。将来の保険料から回収できるなら、商品として成立する場合がある。逆に、解約が早く起きる商品なら、将来回収を待つ前に契約が消え、費用だけが残る。だから事業費の問題では、新契約費、維持費、解約、予定事業費枠、商品別収益性が一緒に問われる。

## 読み方のコツ

事業費の答案は、次の三つで読む。

1. 費用はいつ発生したか。
2. その費用はどの商品・チャネル・契約群団が負担すべきか。
3. 保険料からの回収は、どの期間に見込んでいるか。

この三つが見えると、予定事業費枠や費差損益はただの計算ではなくなる。費用をどこへ帰属させるか、将来回収をどこまで信じるかという経営判断の話になる。
