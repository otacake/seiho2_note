# 06_ソルベンシー

## この単元の芯

ソルベンシーは、通常予測される支払を超えて悪いことが起きても、保険会社が契約を守れるかを見る単元である。責任準備金が「通常想定される将来支払」に備える負債だとすると、ソルベンシーはその外側にある不確実性に備える厚みである。

ここを間違えると、責任準備金と自己資本を混同する。責任準備金を積めば安全、というだけではない。死亡率が悪化する、金利が動く、解約が増える、資産価格が下がる、信用事故が起きる。こうした変化に対して、通常の負債評価を超える余力が必要になる。その余力を見るために、ソルベンシー・マージン比率、実質資産負債差額、早期是正措置、経済価値ベース規制が出てくる。

日本では、従来のソルベンシー・マージン比率に加えて、経済価値ベースのソルベンシー規制の導入が進められてきた。金融庁の有識者会議やIAISのInsurance Capital Standardの議論を見ると、保険負債と資産をより経済価値で見て、リスク量と資本を対応させる方向が強い。試験で「静的評価」「動的評価」「経済価値」が問われるのは、この見方の違いを説明させるためである。

## 暗記事項は何を言っているか

「静的なソルベンシーの検証」は、ある時点の資産、負債、リスク量、自己資本を見て、支払能力があるかを確認する。日本のソルベンシー・マージン比率はこの代表である。これは分かりやすいが、将来の経路までは十分に見ない。

「動的なソルベンシーの検証」は、将来収支分析やストレスシナリオを通じて、時間の中で会社がどう変わるかを見る。今日の比率が高くても、金利低下、死亡率悪化、解約増加が続けば数年後に弱くなるかもしれない。だから、動的検証では将来の剰余、内部留保、配当、資本政策まで見に行く。

「早期是正措置」は、監督当局が会社の状態に応じて介入する仕組みである。ここでは、ソルベンシー指標が単なる説明資料ではなく、監督行動につながる点が重要である。数値が一定水準を下回れば、業務改善命令などの対応が問題になる。

「経済価値ベース」は、資産と負債を市場整合的に見ようとする考え方である。法定会計上の責任準備金は安定性や保守性を重視する。一方、経済価値ベースでは、金利や市場価格の変化が資産・負債にどう反映されるかをより直接に見る。その分、会社のリスク構造が見えやすいが、前提やモデルの妥当性が強く問われる。

## 具体的に見る

低金利の時代に高予定利率契約を多く持つ会社を考える。法定会計では、標準責任準備金や既存の評価ルールに従って負債を積む。一方、経済価値ベースでは、将来キャッシュフローを現在の金利環境で割り引く。金利が下がると、遠い将来の支払ほど現在価値が大きくなる。資産側の債券価格も上がるが、負債のデュレーションが長ければ、負債の増え方が資産を上回ることがある。

このとき、会計利益だけを見ると問題が小さく見える場合がある。しかし経済価値で見ると、会社の純資産が傷んでいるかもしれない。ソルベンシーの答案で、法定会計、経済価値、ALM、ストレステストがつながるのはこのためである。

## 読み方のコツ

ソルベンシーの答案は、まず「何に備える厚みか」を置く。

1. 責任準備金は通常想定される支払に備える。
2. 自己資本やソルベンシー・マージンは、通常を超える悪化に備える。
3. 静的検証は一時点を見る。
4. 動的検証は将来経路を見る。
5. 経済価値ベースは、市場環境とリスク構造をより直接に反映する。

この順に置くと、制度名の丸暗記ではなく、支払能力を見るレンズの違いとして覚えられる。
