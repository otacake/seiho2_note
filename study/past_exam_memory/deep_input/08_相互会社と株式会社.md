# 08_相互会社と株式会社

## この単元の芯

相互会社と株式会社の違いは、会社形態の名前を覚える単元ではない。誰が会社の残余利益を受けるのか、誰のために剰余を使うのか、資本をどう調達するのかを分ける単元である。

株式会社では、株主が出資者であり、剰余の帰属や資本政策は株主との関係を持つ。契約者配当がある場合でも、株主利益と契約者配当の関係を整理する必要がある。相互会社では、保険契約者が社員として会社の構成員になる。だから剰余の分配、基金、非社員契約、有配当・無配当保険の扱いが独特の問題になる。

保険業法上、相互会社は株式会社と並ぶ保険会社の組織形態として扱われる。相互会社では、株式会社の株式にあたる資本調達手段がそのまま使えるわけではない。基金や内部留保の意味が重くなる。試験でこの単元が出るのは、会社形態の違いが、配当、健全性、区分経理、資本政策へ波及するからである。

## 暗記事項は何を言っているか

「相互会社」は、契約者が社員となる会社形態である。ここでいう社員は、従業員ではなく構成員である。剰余の帰属を考えるとき、この点が効いてくる。相互会社で有配当保険と無配当保険を併売する場合、無配当保険の利益が最終的にどこへ帰属するのか、公正・衡平にどう扱うのかが問題になる。

「株式会社」は、株主資本を通じて資本調達しやすい一方、株主利益と契約者保護を整理する必要がある。株主がいるから契約者を軽視してよいわけではない。保険会社である以上、契約者保護と支払能力確保は監督上の中心に残る。

「基金」は、相互会社における資本性のある資金調達手段として理解する。株式会社の株式と同じではない。基金は返済や利息の問題を持ち、相互会社の担保資金や健全性と関係する。

## 具体的に見る

相互会社が無配当保険を売る場面を考える。無配当保険の契約者は、配当を受けない代わりに保険料が低いことを期待する。一方、相互会社全体の剰余は社員である有配当契約者の利益にも関係する。無配当保険から出た利益を、有配当契約者へそのまま還元してよいのか。逆に、有配当契約の剰余で無配当保険の損失を支えてよいのか。ここで区分経理、公正・衡平、セルフサポートの考え方が必要になる。

株式会社の場合は、株主配当と契約者配当が並ぶことがある。株主へ利益を出しすぎれば契約者保護に疑義が出るし、契約者へ還元しすぎれば株主資本の維持が問題になる。相互会社と株式会社の比較は、どちらが良いかではなく、剰余の帰属と資本調達の制約が違うという話である。

## 読み方のコツ

この単元は、次の四つで見る。

1. 会社の構成員は誰か。
2. 剰余は誰に帰属するのか。
3. 資本をどう調達するのか。
4. 有配当・無配当・非社員契約をどう公平に扱うのか。

会社形態の比較表だけを覚えると、答案が浅くなる。配当や区分経理の単元とつなげて読むと、相互会社の論点はかなり覚えやすい。
