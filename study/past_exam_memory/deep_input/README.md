# 生保2 過去問丸暗記 深い理解インプット

このフォルダは、`cards/` や `units/` の暗記答案を置き換えるものではない。暗記答案は原文ベースで残し、ここでは「その答案が何を言っているのか」「なぜ試験でその形で聞かれるのか」を読む。

使う順番は次の通り。

1. このフォルダで単元の芯を読む。
2. `units/` で原文答案と理解メモを確認する。
3. Anki の `whole` で白紙再現する。
4. 崩れる答案だけ `chunk` で段落ごとに直す。
5. 数値・条文・列挙名は `cloze` で最後に確認する。

## 書き方のルール

- 暗記対象の答案文言は、勝手に言い換えない。
- ここで書くのは、答案の前提、制度の役割、具体的な状況、読み方の軸。
- 「例え話」よりも、保険会社で実際に何が起きているのかを優先する。
- 法令・監督・実務基準に関わる説明は、一次ソースに近い資料を確認して書く。
- 古い過去問答案と現行制度がずれる可能性がある箇所は、断定しすぎない。
