# 07_内部管理会計_区分経理

## この単元の芯

内部管理会計と区分経理は、法定会計だけでは見えにくいものを見るための単元である。法定会計は、契約者保護、支払能力、監督上の健全性を見るために強い。一方で、どの商品が価値を生んでいるか、どのチャネルが費用を使っているか、どの契約群団に剰余がたまっているかは、そのままでは見えにくい。

内部管理会計は、経営判断のために数字を組み替える。区分経理は、商品区分や資産区分ごとに損益・資産・負債を分ける。EVやMCEVは、将来利益や経済価値を前に出して、法定会計とは違う角度から会社価値を見る。これらは別々のテーマに見えるが、共通しているのは、平均で見えないものを分けて見るという点である。

金融庁の監督指針でも、区分経理や内部管理の着眼点は、会社の健全性や契約者間の公平性と結びついている。区分経理は単なる社内管理ではなく、配当や商品別採算にも影響する。

## 暗記事項は何を言っているか

「内部管理会計の必要性」は、法定会計の限界から出てくる。法定会計は契約者保護のために保守的で、単年度損益を安定的に見る。その結果、新契約の将来価値、商品別採算、資本コスト、リスク調整後収益性は見えにくい。内部管理会計は、経営が商品改定、販売戦略、資本配賦、撤退判断をするために必要になる。

「区分経理」は、商品や資産のまとまりごとに損益や資産を分ける考え方である。これにより、ある商品区分の損失を別の商品区分の剰余で隠すことを防ぎやすくなる。契約者配当の公正・衡平にも関係する。剰余がどの区分で発生したかが分からなければ、誰へ返すべき剰余かも分からない。

「EV/MCEV」は、既契約から将来生じる利益を価値として見る。法定会計では、新契約時に費用が先に出て赤字に見えやすい。一方、将来利益を考えれば、その新契約は価値を生んでいるかもしれない。EVはこの将来利益を前に出す。ただし、前提、割引率、リスク調整、オプション・保証の評価が結果を大きく左右する。

CFO ForumのMCEV原則では、MCEVを対象事業における株主利益の価値として定義し、将来経験前提、経済前提、オプション・保証、必要資本などを明示的に扱う。試験でMCEVが出るときも、単に「将来利益を見る指標」と覚えるだけでは弱い。どの前提で将来利益を割り引き、どのリスクを調整して、どの資本を拘束しているのかまでが論点になる。

## 具体的に見る

会社全体では黒字だが、ある商品区分だけを見ると継続的に費差損が出ているとする。会社全体の法定損益だけを見れば問題は見えない。しかし、その商品の新契約費が高く、解約も早く、将来回収が難しいなら、経営としては商品改定や販売抑制を考える必要がある。

また、有配当保険と無配当保険を同じ会社が売っている場合、無配当保険から出た利益をどのように扱うかが問題になる。相互会社なら最終的な剰余の帰属にも注意が必要である。区分経理が問われるのは、こうした内部補助や契約者間の不公平を見えやすくするためである。

## 読み方のコツ

この単元は、次の問いで読むとつながる。

1. 法定会計では何が見えないのか。
2. 商品別・区分別に分けると、どの損益や剰余が見えるのか。
3. その数字は、配当、商品改定、販売戦略、資本配賦のどれに使うのか。
4. 将来利益を見るとき、どの前提が結果を動かしているのか。

内部管理会計は、法定会計の否定ではない。法定会計で守るべきものを守ったうえで、経営判断に使える粒度へ数字を分解する道具である。
