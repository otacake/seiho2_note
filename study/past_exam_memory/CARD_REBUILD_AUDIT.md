# 過去問丸暗記カード 再設計監査

## 先に確認したダメだった点

- 長文答案を `whole` にそのまま入れ、1枚で再現するには重すぎるカードがあった。
- `chunk` が文字数で機械分割され、同じカード内に別の見出し・別の小問が混ざっていた。
- `小問` や `穴埋め` 系で、覚えるべき語句・数値・処理が黒字のまま残るカードがあった。
- 逆に、文章全体を抜きすぎると、何について答えているのか読めない穴埋めになる危険があった。
- cloze が一部単元に偏り、生命保険会計以外の数値・条文・列挙確認が不足していた。

## 今回の修正方針

- `whole`: 原問題から白紙再現する入口。長文・小問羅列・答え漏れリスクがあるものは答案骨子にして、詰め込みを避ける。
- `chunk`: 見出し、列挙、小問ごとに切り直し、原文答案の再現単位にする。
- `cloze`: 数値、条文、制度名、列挙名、定義語句だけを抜く。問題文として必要な文脈は残す。
- `units`: Ankiに混ぜない理解メモと、原文答案の正本を保持する。
