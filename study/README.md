# Study Workspace

このディレクトリは、`生保2` 合格用の学習資産を GitHub で管理するための作業領域です。

## 役割

- `first_part/`
  第I部のインプット資料。学習単位は「1問1答」。
- `anki/`
  Anki 取り込み用の説明とエクスポート先。

## 基本方針

- 第I部は `ワークブック + 過去問` を単元別に再編し、問題と基準答案を 1 対 1 で管理する。
- ただし、丸暗記の前提でも `要するに何を聞いているか` と `そもそもの問題意識` は必ず別欄に置く。
- Anki の想起は「問題を見て全文を書く」前提で設計する。
