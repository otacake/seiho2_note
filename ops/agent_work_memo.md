## このメモの目的

- 以後の作業では、まずこのメモを見てから着手する

## 固定指示

- `基準答案` は、原則として `過去問の解答があるならその文言を優先して載せる`
- 過去問の解答がない場合や、複数資料をまとめた方がよい場合でも、`既存ノート` `法令` `一次資料` の文言をできるだけ崩さない
- `ざっくり説明` `要するに何か` `問題意識` は、基礎から、直感的に、行間を残さず書く
- `詳細説明` は必須と考える。`基準答案` の各行の前提、因果、制度趣旨、なぜその文言になるかを、行間がなくなるまで埋める
- 説明は `何を守りたいか` `何が起点で` `何を通じて` `どの指標が` `なぜ問題になるか` まで書く
- 説明文は人が読んで自然な日本語にする。少なくとも、機械的な定型表現、太字頼みの強調、抽象語だけの説明は避ける
- 日本語の説明文は、必要に応じて `ja-natural-humanizer` の方針に寄せて推敲する
- 法令・係数・区分は、古いノートをうのみにせず、必ず現行一次資料を確認する
- `study/first_part/` では、強調のための太字に依存しない

## 2026-04-05 ソルベンシー経済価値パートの再深掘り

- ユーザー要請
  - `05_ソルベンシー.md` の `経済価値` のところを、本当にゼロから読めるようにしてほしい
  - 読み手はアクチュアリーなので、算式や投資理論、デリバティブの考え方は使ってよい
  - ただし、式や理論を置くだけではなく、その意味を全部丁寧に説明する
- 今回やったこと
  - `価値とは何か` を起点に、`PV = Σ CF_t / (1 + r_t)^t` から入り、経済価値が `将来キャッシュフローを今日へ引き直す考え方` であることから書き直した
  - 保険負債は `長期で不確実なキャッシュフロー` なので、債券よりはるかに評価が難しいことを、死亡・長寿・解約・費用・保証の不確実性に分けて説明した
  - 経済価値の世界でのリスクは、`価格が動くこと` ではなく `S = A - L` の差額が傷むことだと置き直し、デュレーションとコンベクシティの式で金利リスクの見方を補強した
  - `市場整合的評価` では、`無裁定` `複製` `リスク中立価格付け` の順で説明し、単なる「時価っぽい評価」ではないことを明示した
  - さらに、最低保証つき変額商品の例として `max(S_T, K) = S_T + max(K - S_T, 0)` を入れ、保証部分が実質的にプット・オプションであることを説明した
  - この追記で、保証価値が `株価` `ボラティリティ` `金利` `時間` で動くこと、なぜ平均キャッシュフローの割引だけでは足りないこと、なぜ確率論的シナリオやオプション評価が要ることまでつないだ
  - あわせて、`市場でかなりヘッジできる部分` と `契約者行動や費用など複製できずに残る部分` を切り分け、前者は市場整合的評価、後者は最良推定やリスク・マージンで受けるという役割分担を明確化した
  - `なぜリスクフリー金利なのか` では、`保険負債そのものの価値` と `会社固有の運用戦略` を混ぜないためだと書き直した
- 今回の判断
  - 経済価値規制は、制度名だけ追うと分かった気になりやすいが、`何を負債で見て` `何を資本側で見るか` の線引きが腹落ちしないと本質がつかみにくい
  - そのため今回は、制度説明の前に `価値とは何か` `オプションとはどんな約束か` `会社固有の運用方針をなぜ負債評価へ混ぜないのか` を順番にほどく構成へ寄せた
  - 文体は引き続き `ja-natural-humanizer` に寄せ、抽象語だけで押し切らず、読み手が一段ずつ納得できる柔らかい説明を優先した
