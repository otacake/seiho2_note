# 入力テンプレート

必要な欄だけ埋めれば十分です。空欄があっても使えます。

[依頼種別]
新規作成 / 推敲 / AIくささ除去 / 文体合わせ / 要約

[原文またはメモ]

[読者]

[目的]

[媒体]
メール / 社内文書 / ブログ / レポート / 説明文 / その他

[文体]
です・ます調 / である調 / 任せる

[専門性]
専門用語を残す / 初出で説明する / 一般向けに言い換える

[絶対に残す情報]
数値、固有名詞、条件、期限、例外など

[避ける語]
誇張語、社内で嫌われる語、業界用語など

[参考にしたい文体]
自分の過去文、社内文書、既存記事などのファイルパスまたは本文
